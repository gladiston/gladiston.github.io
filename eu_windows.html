Eu e o Windows

Especialista em Windows Server, Deployments e Segurança de Políticas.

Desde o Windows 3.0 e NT 3.5, acompanho a evolução da plataforma Microsoft. Minha abordagem é técnica: utilizo o Windows Terminal e PowerShell para gerenciar servidores com a mesma precisão que uso no Linux.

Windows Server & Hypervisor

Gestão de edições com GUI e Windows Core. Experiência em recuperação de bases corrompidas e virtualização com Microsoft Hypervisor.

Segurança & GPO

Implementação de políticas rígidas: controle de execução de software, bloqueio de dispositivos removíveis e quiosques de segurança para ambientes multiterminal.

Automação via Scripts

Desenvolvimento de ferramentas de manutenção e backup proprietárias usando VBScript e arquivos CMD, incluindo o uso estratégico de RunAs para elevação de privilégios segura.

MSSQL Server

Especialista em SQL Server desde a versão 6.5. Proficiente em escrita de scripts DML/DDL/PSQL e manutenção de grandes bases de dados.

A integração do WSL2 trouxe o melhor dos dois mundos, permitindo o uso de ferramentas Linux de forma nativa no ambiente Windows.
