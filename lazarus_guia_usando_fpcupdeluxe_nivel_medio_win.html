Se você seguiu a instalação via instalador padrão e algo deu errado, ou quer algo mais moderno e automatizado, essa é a segunda via para o Windows. O fpcupdeluge (Free Pascal Compiler Updated Deluxe) é uma ferramenta gráfica que automatiza o download e compilação do FPC e do Lazarus.

1. Vantagens do fpcupdeluge

Diferente do instalador oficial, o fpcupdeluge permite que você tenha várias versões do Lazarus instaladas simultaneamente em pastas diferentes (instalações portáteis). Isso é excelente para manter um projeto antigo em uma versão estável e testar novos recursos na versão de desenvolvimento (Trunk).

2. Download da ferramenta

Baixe o executável para a sua plataforma (Windows i386 ou x86_64) no repositório oficial:

https://github.com/LongDirtyAnimall/fpcupdeluxe/releases

todo: gravar imagem assets/img/fpcupdeluge_download.png

3. Preparando o ambiente

Antes de iniciar, crie uma pasta onde o Lazarus será instalado. Recomendo algo simples como C:\Lazarus_Stable ou C:\Lazarus_Trunk. Nunca instale em pastas com espaços no nome, como "Program Files", para evitar erros de compilação.

4. Configuração e Instalação

Ao abrir o programa, ele apresentará diversas opções. Siga estes passos para uma instalação padrão estável:

  • No campo Install directory, selecione a pasta que você criou no passo anterior.
  • Certifique-se que o FPC version e Lazarus version estão marcados como "Stable" (ou selecione a versão específica que deseja).
  • Clique no botão Install/Update FPC + Lazarus.
todo: gravar imagem assets/img/fpcupdeluge_interface.png

O processo pode demorar, pois ele fará o download de todo o código fonte e compilará os binários em sua máquina. Acompanhe o log na janela preta (terminal) que o programa abre.

5. Criando o atalho

Ao final da instalação, o programa perguntará se deseja criar um atalho na área de trabalho. Aceite para facilitar o acesso àquela instância específica do Lazarus.

Dica: Se você precisar de cross-compiling (compilar para Linux ou Android a partir do Windows), o fpcupdeluge facilita muito a instalação dos "Cross-Compilers" através da aba Setup Cross.

CONCLUSÃO

O fpcupdeluge é uma ferramenta poderosa que retira o peso da configuração manual de caminhos, sendo ideal para quem precisa de flexibilidade e quer gerenciar múltiplas versões da IDE sem conflitos no registro do Windows.
