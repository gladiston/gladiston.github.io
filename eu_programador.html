Minha História com o Código

Minha paixão pela programação começou com a Linguagem de Máquina (Z80 e 6502) e o BASIC para sistemas 8-bits. Aprendi cedo que para extrair o máximo de máquinas como o Apple ][, CP-400 e MSX, era necessário dominar endereçamento de memória e VRAM para criar gráficos e sprites eficientes.

Profissionalmente, evoluí do Clipper e DBASE para arquiteturas modernas, focando sempre na robustez e escalabilidade do código.

Pascal & Delphi

Domínio total de Delphi, Lazarus e Turbo Pascal. Expertise em DataSnap (Multitier) e Microsserviços Web.

Automação & Backend

Python para automação de processos críticos e Bash para gestão de servidores Linux.

Web & Fullstack

Desenvolvimento frontend com JS (ES6), HTML5/CSS3 e backend procedural em PHP.

Bancos de Dados

SQL ANSI-2002. Experiência com MSSQL Server, Oracle, PostgreSQL, Firebird e MySQL.

Atualmente aprofundando conhecimentos em Node.js e Vue.js para aplicações Web de última geração.
