A instalação manual com Lazarus+FPC no Linux permite uma instalação homeuser. Isso significa que você pode instalar e usar a IDE sem permissões administrativas (root), mantendo tudo dentro da sua pasta HOME.

Antes de começar, certifique-se de ter o compilador FPC instalado via repositório:

Instalação no Linux: Instalar o FPC a partir dos repositórios

Resolvendo Dependências

Para corrigir avisos sobre o módulo pk-gtk-module, execute:

sudo -i
echo "/usr/lib64/gtk-3.0/modules" >> /etc/ld.so.conf.d/pk-gtk.conf
echo "/usr/lib64/gtk-2.0/modules" >> /etc/ld.so.conf.d/gtk2.conf
ldconfig

Para dependências de som e QT5:

Debian/Ubuntu/Mint:

sudo apt install -y libqt5pas-dev libqt5pas1 libcanberra-gtk-module libcanberra-gtk3-module

Fedora:

sudo dnf install -y qt5pas-devel qt5-qtbase-devel libcanberra-gtk2 libcanberra-gtk3
