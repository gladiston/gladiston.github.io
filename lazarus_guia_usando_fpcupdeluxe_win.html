Usando fpcupdeluxe (nível médio)

O fpcupdeluxe permite uma instalação homeuser, sem necessidade de permissões de administrador. O programa compila o FPC e o Lazarus otimizados para o seu processador.

Visite a página de releases:

https://github.com/LongDirtyAnimAlf/fpcupdeluxe/releases

Passo #1: Execute o fpcupdeluxe.exe. Aceite a telemetria se desejar ajudar os desenvolvedores.

Passo #2: Selecione a versão fixes para FPC e Lazarus e clique em Install/update FPC+Laz.

Passo #3 (Opcional): No botão Setup+, defina critérios como o alvo das compilações.

Configuração Importante

Use sempre o atalho Lazarus_fpcupdeluxe, pois ele contém o parâmetro essencial --pcp:

--pcp=C:\pasta\de\instalacao\do\lazarus\configs
