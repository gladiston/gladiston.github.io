Introdução

O menu Tools do Lazarus permite que acrescentemos ferramentas externas para que não precisemos sair da IDE para abrir programas auxiliares ou sites de referência. Isso agiliza o fluxo de trabalho, centralizando utilitários no ambiente de desenvolvimento.

Configurando Ferramentas Externas

Para adicionar um programa, acesse Tools -> Configure External Tools. Você verá uma janela para gerenciar sua lista de utilitários:

Ao clicar em Add, você deve preencher os campos essenciais. Por exemplo, para adicionar o DBeaver (um excelente gestor de bancos de dados):

As informações de "Program Filename" e "Parameters" podem ser copiadas dos atalhos que você já usa no sistema.

Dica de Produtividade: Use o campo Key para definir um atalho global (ex: Ctrl+Shift+D). Assim, você abre sua ferramenta favorita instantaneamente sem usar o mouse.

Localizando Programas no Linux

Se você usa Linux e não sabe o caminho exato do executável, use o comando whereis no terminal:

whereis firefox
# Saída: /usr/bin/firefox

whereis flamerobin
# Saída: /usr/local/bin/flamerobin

Para programas instalados via Flatpak, a chamada é um pouco diferente. Primeiro identifique o ID do aplicativo:

flatpak list | grep VLC
# Saída: org.videolan.VLC

No Lazarus, o comando seria flatpak e o parâmetro run org.videolan.VLC.

Sugestões de Ferramentas (Windows)

Aqui estão alguns utilitários que considero essenciais no meu dia a dia:

  • DBeaver: Gestão de múltiplos bancos de dados.
  • FlameRobin: Utilitário leve para Firebird.
  • LRS Explorer: Explorador de recursos do Lazarus.
  • Mapa de Caracteres: %windir%\system32\charmap.exe (Nativo do Windows).
  • Android Asset Studio: Excelente para gerar ícones.

Conclusão

O menu Tools e as ferramentas externas permitem centralizar programas e atalhos na IDE. Use Tools | Configure External Tools para adicionar DBeaver, FlameRobin, documentação e outros utilitários, e defina atalhos de teclado para acesso rápido. Isso reduz trocas de janela e acelera o desenvolvimento.
