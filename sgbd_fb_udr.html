UDR: User Defined Routines

Confira abaixo uma coleção de recursos e informações sobre UDRs (User Defined Routines) no FirebirdSQL. As UDRs permitem estender o Firebird com funções customizadas desenvolvidas em linguagens compiladas como Pascal e C++.

O que são UDR (User Defined Routines)?

A partir da versão 3.0, o Firebird introduziu as UDRs, substituindo o antigo conceito de UDFs. As UDRs são plugins integrados que rodam dentro do motor com segurança e contexto de transação, permitindo cálculos complexos, integração de sistemas e criptografia customizada com alta performance.

Lucene FTS UDR

Biblioteca UDR que implementa procedimentos e funções de busca full-text para Firebird SQL usando o poderoso motor de busca Lucene++. Permite realizar buscas de texto completo com alta performance e recursos avançados de análise de texto, incluindo suporte para múltiplos idiomas e analisadores customizados.

JSON UDR library for Firebird

Biblioteca UDR desenvolvida em FreePascal para trabalhar com JSON no FirebirdSQL 3.0. Funciona em Windows e Linux, com licença MIT (100% livre e open source). Permite manipular e processar dados JSON diretamente no banco de dados Firebird através de funções UDR.
