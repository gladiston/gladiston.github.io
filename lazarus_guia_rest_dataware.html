Introdução

O Rest Dataware (RDW) é uma badalada ferramenta para criar e consumir serviços de API REST. Pode ser instalada e usufruída tanto no Lazarus-IDE como também no Delphi Community Edition.

Se pretende consumir serviços REST, esta é uma excelente framework. No entanto, se não for utilizar esses recursos de imediato, evite a sua instalação, pois ela é pesada e depende dos componentes Indy.

Instalação e Configuração

No Delphi Community, a instalação pode ser feita via GetIt ou repositórios oficiais. Já no Lazarus-IDE, o RDW está disponível no Online Package Manager (OPM).

Importante: Antes de instalar o RDW no Lazarus, instale manualmente o pacote Indy. Isso evita conflitos na automação do OPM e garante um ambiente mais estável.

Conclusão

O RDW é uma ferramenta poderosa que se destaca pelos seus componentes visuais e facilidade de uso em RAD. Comparado ao Horse, o RDW oferece uma estrutura mais completa para quem prefere trabalhar com componentes em vez de apenas código minimalista. Ambos os frameworks são excelentes e essenciais no arsenal de um desenvolvedor Pascal moderno.
