Início Guia de Sobrevivência

O objetivo desta página é ajudar iniciantes e programadores avançados que estejam interessados em aprender Pascal usando a IDE de programação do Lazarus. Os artigos estão dispostos para um aprendizado progressivo.

1. Instalação no Windows

  • Usando Instalador padrão (nível fácil)
  • Usando como instalador o fpcupdeluxe (nível médio)
  • Integrando o editor de código com ajuda on-line (Help)

2. Instalação no Linux

  • Usando fpcupdeluxe (nível médio)
  • Lidando com a configuração do Fppkg
  • Integrando o editor de código com ajuda on-line (Help)

3. Essencial, mas não básico

  • Introdução ao IDE Lazarus
  • A prova de balas
  • Escolhendo um idioma
  • Gerenciador de projetos (Project Manager)
  • Online Package Manager (OPM/GPO)

4. Personalizando a IDE

  • Ferramenta “Desktop” – O que é e como usar
  • Docagem básica e do editor de formulário
  • Em busca das janelas perdidas
  • Usando o inspetor de projeto (Project inspector)
  • Atalho Ctrl+H para componentes não visuais

5. Depuração de erros (debug)

  • Dê olá ao FpDebug
  • Trocando o FpDebug para o conhecido gdb
  • Controle de qualidade básica ao projeto

6. Instalação de pacotes

  • ZeosLib e DCPcrypt
  • FPSpreadsheet e HtmlViewer
  • UserControl e ACBr

7. Gerador de relatórios

  • Relatórios com Fortes Report
  • Relatórios com LazReport
  • Relatórios com o fpReport

8. Produtividade

  • Atalhos úteis
  • Salvando configuração da IDE
  • Usando GIT em projetos FPC/Lazarus
  • Criando Build Modes
  • Refatoração de código usando a IDE

9. Componentes rotineiros

  • Usando Zeos – Conexão com base de dados
  • ComboBox para valores complexos
  • Tchau ShowMessage, Olá TaskDialog
